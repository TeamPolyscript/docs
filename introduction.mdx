## What is Polyscript

Polyscript is an algorithmic trading platform for [Polymarket](https://polymarket.com). Write Python strategies, backtest with historical data, and deploy ones that run 24/7 — all from one workspace.

## Core features

<AccordionGroup>
  <Accordion title="Write strategies in Python">
    A small SDK gives you access to live and historical Polymarket data. Decide your own positioning, ordering, and risk rules.
  </Accordion>

  <Accordion title="Backtest with historical data">
    Replay strategies against 100 days of Polymarket tick data. Tune parameters before going live.
  </Accordion>

  <Accordion title="Deploy strategies that run 24/7">
    One-click deploy. The platform handles uptime, retries, and observability.
  </Accordion>

  <Accordion title="Monitor in real time">
    Position, PnL, and order book state surfaced in the web console.
  </Accordion>
</AccordionGroup>
